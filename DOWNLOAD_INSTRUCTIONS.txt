==========================================================================
HOW TO GET THE ZOHO GYM CAMPAIGNS SKILL ON YOUR MAC
==========================================================================

OPTION A: COPY FROM THIS SESSION (EASIEST)
-------------------------------------------

1. All your skill files are located at:
   /Sessions/Cowork/outputs/zoho-gym-campaigns/
   
2. Copy the entire "zoho-gym-campaigns" folder to your Mac:
   ~/Projects/zoho-gym-campaigns

3. The folder contains:
   - main.py (main script)
   - google_sheets_reader.py (Google Sheets integration)
   - zoho_campaigns_client.py (Zoho integration)
   - campaign_generator.py (Campaign creation)
   - templates.py (Email templates)
   - config.txt (Your configuration - ALREADY SET UP!)
   - requirements.txt (Python dependencies)
   - run.sh (Automation script)
   - Various test files


OPTION B: COPY FILES MANUALLY
------------------------------

From Finder:
1. Go to: ~/Library/Application Support/Claude/
2. Find the Cowork session folder
3. Navigate to: outputs/zoho-gym-campaigns
4. Copy the entire folder to ~/Projects/


THEN RUN IT
-----------

Once you have the folder in ~/Projects/zoho-gym-campaigns:

1. Install dependencies:
   pip3 install -r requirements.txt

2. Run the skill:
   cd ~/Projects/zoho-gym-campaigns
   source config.txt
   python3 main.py

That's it! 🚀
